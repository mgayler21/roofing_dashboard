# Roofing Analytics CLI Dashboard

A Python command-line analytics tool for roofing job tracking.

## Features
- Loads job data from CSV
- Calculates revenue, cost, and profit
- Shows best and worst jobs
- Displays monthly profit totals
- Exports reports to CSV
- Visualizes monthly profit with Matplotlib
- Lets users add new jobs through a menu

## Tech Used
- Python
- CSV
- Matplotlib

## Files
- roofing_dashboard.py
- roofing_jobs.csv
- roofing_jobs_report.csv
