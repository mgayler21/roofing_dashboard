# Roofing Analytics CLI Dashboard

A Python command-line analytics tool for tracking roofing jobs, calculating profit metrics, exporting reports, and visualizing monthly performance.

This project was built to combine real roofing industry workflow with Python data analysis.

---

## Features

• Load roofing job data from a CSV file  
• Calculate revenue, cost, and profit  
• Identify highest and lowest profit jobs  
• Display monthly profit totals  
• Export reports to CSV  
• Generate profit charts with Matplotlib  
• Add new jobs through a CLI menu system  

---

Matplotlib Data Visualization
